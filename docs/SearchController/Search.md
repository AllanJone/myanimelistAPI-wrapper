## SearchForAnime
Searhes myanimelist for anime based on the specified query.

### Parameters

| Type | Name | Description |
| ---- | ---- | ----------- |
| System.String | searchQuery | The query to search for. |

### Returns
[AnimeSearchResult] Object which contains a List of [AnimeSearchEntry].
