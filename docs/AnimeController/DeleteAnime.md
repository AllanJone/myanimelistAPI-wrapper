## DeleteAnime
Deletes existing anime from user's list.

### Parameters

| Name | Description |
| ---- | ----------- |
| [System.Int32] animeId | The anime entry ID to delete from the list. |

### Returns
[System.String] represnting the state of deleting "Deleted" or detailed error message.
